Overview

Two additional modules were carried out for the new "StratAdd" package. These two functional modules are RunExportation which exports the characteristics of biodiversity in the form of a histogram, as well as a second one, Export character by tree which consists in exporting the characteristics of taxa with the tree in a text file.

Modules

The package contains two modules:

  • RunExportation
  • Export character by tree

    RunExportation


    You can find the button in the panel on the left like on the following picture.


    Click on the button to select it. Then go on the tree to select the branch which you wish to know the histogram of biodiversity.
    When you can select it, the branch appears in white.


    Then you click on it.


    A window appears. Choose 1 or all the indices to export in the excel file and press "OK".


    Another window appears. Click on "save" to choose the destination.



    Select your folder destination, change the name of the file and press "save". (The picture below is a sceenshot from a Mac OS)


    Click on "OK" to quit.


    All data are in the output.xls file.



    The output is mostly interesting if the tree includes extinct species placed in a stratigraphic context using the Stratigraphic tools for Mesquite (Josse et al. 2006). In that case, the evolution of known biodiversity of a clade through the geological ages can be displayed as a histogram.




    Export character by tree


    To find the "Export Character by Tree" option, select the Tree menu, then Utilities, as you can see on this screenshot.


    You have the possibility of choosing the Taxa and the characters with the matrix of characters if several are existing (cf picture below) for which a table will be exported.


    Select the characters you want to export in the excel file.



    The character data in the output file is sorted by taxa in the same order as in the tree in the active tree window. The output can then be transferred to standard statistical packages to perform various analyses, such as PVR analysis with variance partitioning (Desdevises et al., 2003; Cubo et al., 2005), or a Mantel test (Mantel, 1967; Laurin, 2004).



    Installation

    To install the StratAdd package, you must add the "StratAdd" directory in the mesquite folder within Mesquite_Folder. Then launch the Mesquite application.
